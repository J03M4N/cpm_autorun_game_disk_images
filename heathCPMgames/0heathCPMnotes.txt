Workflow: cpm2204s.td0 is a setup disk. Use HxC to convert a copy of HOS8719A.IMD to .MFM. Use .td0 as flop1 and .MFM as flop2, start emulator, setup runs automatically, choose 96 for drive density. select original .IMD for file source. .MFM is now the boot disk. Use HxC to convert .MFM to .IMG, edit .IMG with CPMtoolsgui, convert .IMG with 'Load Raw image' in HxC using above values, export as .MFM (R/O .IMD doesn't work with this system without errors).
