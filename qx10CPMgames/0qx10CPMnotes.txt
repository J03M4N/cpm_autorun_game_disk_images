	seclen 256
	tracks 82

	blocksize 2048

Source cpm boot disk image from Dave's old computers.

Workflow: load source .IMD in HxC, export .IMG, use CPMtoolsGUI to edit .IMG, use 'load raw image' in HxC with values from above, export as .IMD.

Autostart: There is a program on boot disk called 'AUTOST.COM' that you can use to set an autostart. 
You must first reload the boot disk through mame menu to force the disk read option dialogue, select "read from this image and write to another". Use backspace to rename to .MFM file and save as .MFM, then set autostart in CP/M using AUTOST.COM. Your final file is an .MFM. use HxC to load .MFM and export to .IMD. After this it is easy to find the autostart with hex editor. A whole block must be copied to zero out the garbage.
SUBMIT PROFILE works as startup, PROFILE.SUB must be padded, extract and edit the existing one.

MAME276 command line:

mame64 qx10 -flop1 "[path to disk image]" -window -skip_gameinfo
