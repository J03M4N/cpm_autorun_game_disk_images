Source cpm boot disk image from MAME software list.

Workflow: CPMtoolsGUI can directly edit .D80 files, no conversion required.

Autorun: Hexedit modified the cpm22 bootdisk to autorun SUBMIT.COM PROFILE.SUB (like CP/M 3). Edit PROFILE.SUB to setup autorun.

MAME276 command line:

can be run with "pet8032", or just "softbox" both look very different.

mame pet8032 -window -nomaximize -skip_gameinfo -natural -ieee11 softbox -flop1 "[path to softbox CP/M]" -autoboot_delay 1 -autoboot_command load\"k\",8\n\n\n\n\n\nrun\n

or

mame softbox -window -nomaximize -skip_gameinfo -natural -flop1 "[path to softbox CP/M]"
