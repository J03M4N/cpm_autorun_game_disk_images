Terminal: ADM-3A mostly:

Cpmtools diskdef:

diskdef bw12
  seclen 256
  tracks 40
  sectrk 18
  blocksize 2048
  maxdir 64
  skew 1
  boottrk 2
  os 2.2
end

HxC raw .IMG:

40 tracks
1 side
18 sectors
256 sector size
0 sector id
1 interleave
0 skew
0 format value
255 gap3
0 pregap

Source cpm boot disk image from MAME software list.

Workflow: load IMD in HxC, save .img, xfer files onto .img with cpmtoolsgui, 'load raw image' in HxC with values from above, export .IMD.

Autostart: Autorun is built-in. Rename .COM file to AUTORUN.COM.
Image can be hexedited to autorun BASIC by renaming MBASIC.COM and the *.BAS file to equal 6 letters (+ a space = 7) ie "MB MENU" and hexedit replacing AUTORUN.

MAME276 command line:

mame bw12 -flop1 "[path to disk image]" -window -nomaximize -autoboot_delay 5 -autoboot_command \n\n\n
